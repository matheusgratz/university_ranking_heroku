Where do data come from?

Universities Rankings Data

The rankings data was obtained by web scraping the The world university rankings site. The result was a set of 1526 observations that where placed in a Pandas dataframe.

Universities Locations

In order to obtain the location (lat, long) of the Universities a Google Geocoding request was done. The result was a set 1524 universities that where also placed in a Pandas dataframe.

A different plugin

To build the data table the Tabulator plug in was used to render the table from a javascript file. The advantage is that this plug in allows control on pagination, downloading the table, sorting columns, amog other things.
